🔐 OAuth 2.0 — Google Identity Services
Cómo obtener su Client ID:
1. Vaya a console.cloud.google.com
2. Cree un proyecto → APIs & Services → Credentials
3. Create Credentials → OAuth client ID → Web application
4. En Authorized JavaScript origins agregue el origen donde aloja este archivo (ej. https://su-dominio.com o http://localhost)
5. Habilite la Google Sheets API en APIs & Services → Library
